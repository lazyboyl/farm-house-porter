# 获取我的地址列表
## 获取我的地址列表接口地址
    /address/getMyAddressList
## 获取我的地址列表接口入参
    无
## 获取我的地址列表成功返回
    {
      "code":200,
      "msg":"获取我的地址列表成功",
      "obj":{
         "rows":[
           {
             "addressId":"str",// 地址流水ID
             "addressName":"str",// 详细地址
             "name":"str",// 联系人
             "mobile":"str",// 手机号码
             "defaule":"str" // 是否为默认收货地址【1：默认；0：不默认】
           },
            ...
           ]
       }
     }
## 获取我的地址列表失败返回
    {
      "code":400,
      "msg":"获取我的地址列表失败",
      "obj":null
     }
